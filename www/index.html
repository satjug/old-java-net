Next Meeting--6 September

Title: Fast Code Eclipse Plugin

Using the fast code eclipse plug-in can tremendously speed up development. The generated code can give you a head start. It can help you create interface implementation pair in a spring application. It also has a nice unit test generator and a very useful template mechanism. It does not replace the need for the spring tools suite, but it can be a nice complement to it. Also the template mechanism does not replace the eclipse built-in code templates. It can, however, be a useful complement.

Speaker: Gautam Dev

Gautam Dev is a software engineer with more than ten years experience in various industries. Gautam obtained Ph.D in physics from a well known university in USA, SUNY, Stony Brook and received M.Sc (equivalent to MS) from a well known university in India, I.I.T, Kanpur. Right now he is working on the Fast Code open source Eclipse plugin project. Details about this project can be found at http://fast-code.sourceforge.net/.
